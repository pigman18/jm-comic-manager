# 分类筛选 GET /categories/filter

## 接口参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | int | 是 | 页码 |
| `o` | string | 是 | 排序方式 |
| `c` | string | 否 | 分类 slug。不传或传空串 = 全部分类(A漫)；传 `slug_subslug` 格式筛选到子类 |
| `order` | string | 否 | 固定传空串 `""`；传其他值无效 |

## 排序方式 (`o`)

| `o` 值 | 含义 | 说明 |
|--------|------|------|
| `""` (空串) | 最新发布 | 仅在 A漫(全部分类) 下可用 |
| `"tf"` | 最多爱心 | 所有分类/子类通用 |
| `"mv"` | 总排行 | 所有分类/子类通用 |
| `"mv_m"` | 月排行 | 所有分类/子类通用 |
| `"mv_w"` | 周排行 | 所有分类/子类通用 |

## 参数组合全矩阵

### 大类

共 11 个大类，其中 4 个无小类、7 个有小类。

### 无小类的大类

c 参数=该大类 slug（最新A漫的 c 省略）

| 大类 slug | 小类 | c 值 |
|-----------|------|------|
| `""` (最新A漫) | (无) | 不传 |
| `"hanmansfw"` (一般向韩漫) | (无) | `hanmansfw` |
| `"another_cosplay"` (Cosplay) | (无) | `another_cosplay` |
| `"3D"` (3D) | (无) | `3D` |
| `"禁漫汉化组"` (禁漫汉化组) | (无) | `%E7%A6%81%E6%BC%AB%E6%B1%89%E5%8C%96%E7%BB%84` |

每个无小类大类产生 **5 种组合**（5 种排序，最新A漫的 5 种排序均有效）。

**例：** `?c=hanmansfw&o=mv` — 一般向韩漫 总排行

## 输出格式 (`categoriesReducer.cateFilterList`)

```typescript
{
    list: Array<{
        id: number;
        name: string;
        image: string;       // 封面 URL
        author: string[];
        tags: string[];
        total_views: string;
        likes: string;
        // ...其他字段
    }>;
    tags: string[];          // 可用标签（与当前分类相关）
    total: number;           // 总结果数
    search_query: string;
}
```

## 注意

- 子类 slug 通过 `c = category_slug + '_' + sub_category_slug` 拼接
- 最新A漫 (`c=""` 或 `c` 不传) 时，排序 `o=""`(最新) 和 `o="mr"` 等价
- 非 A漫 的 大类/子类 下，排序 `o=""`(最新) 不可用（回退到默认排序）
- `c` 传 `"0"`（字符串零）= 按 ID 0 过滤，和 `c=""`(全部分类) 结果不同（A漫 ID 0 的漫画集合 vs 全部漫画）
