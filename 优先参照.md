# 优先参照（新 session 必须先读）

## 页面布局模式

### 有过滤区/操作区的页面（如 SerialPage）
```html
<div class="jmz-page jmz-serial-page">
  <div class="jmz-serial-header"><!-- 操作区：过滤条件、分页等，flex-shrink: 0 --></div>
  <div class="jmz-serial-main" ref="mainScrollRef"><!-- 内容区：列表/卡片，flex:1; overflow-y:auto --></div>
</div>
```
- `jmz-serial-page`: `flex:1; display:flex; flex-direction:column; min-height:0`
- `jmz-serial-header`: `flex-shrink:0; margin:12px`
- `jmz-serial-main`: `flex:1; min-height:0; overflow-y:auto; padding:0 12px`

### 操作区内部再分区的，多个 jmz-panel 上下叠
每个 panel 用 `jmz-panel jmz-panel--pad`，自动有 `margin-bottom: 12px` 隔开。

### 评论区（操作区固定 + 内容区滚动）
操作区放顶部（含 form + 分页），内容区 scrollable，结构同 serial-page。

## 加载状态

### 有漫画卡片的页面 → 骨架屏（skeleton card）
`v-for="i in pageSize"` 生成 `jmz-skel-card`，参考 CatalogPage 实现。

### 无漫画卡片的页面 → 居中大转圈
```html
<div style="display:flex;align-items:center;justify-content:center;min-height:200px">
  <n-spin size="large" />
</div>
```
用于 ForumPage、HomePage 等非漫画列表页。

## 封面加载动画（参考 ComicCard）

### 必须的 CSS：父容器要有高度
```css
.jmz-card-cover-wrap { /* 或任何封面容器 */
  position: relative;
  aspect-ratio: 3 / 4;  /* 关键：没有这个 spinner 不可见 */
  overflow: hidden;
  background: #2a2a30;
}
```

### 必须的模板模式
```html
<div class="cover-wrap">
  <div v-show="!coverLoaded" class="jmz-cover-spinner">
    <n-spin size="small" />
  </div>
  <img ref="coverRef" src="..." @load="coverLoaded = true" @error="coverLoaded = true" />
</div>
```

### 必须的缓存图片处理
```js
const coverRef = ref<HTMLImageElement | null>(null)
watch(coverRef, (el) => {
  if (el && el.complete && el.naturalWidth > 0) coverLoaded.value = true
})
```
- 不用 `new Image()` 预加载
- 不用 opacity 过渡（除非真的需要）
- 容器必须要有 aspect-ratio 或固定高度

## 图片不显示时先排查
1. CSS：父容器有宽高吗？spinner 是 `position:absolute;inset:0`，容器高度为 0 则不可见
2. 同类实现对比：grep 现有实现（ComicCard/CatalogCard），看 CSS 差异
3. JS 逻辑不生效时先查 CSS 布局，不要堆更多 JS

## 分页规则
- API 返回 `total` → 传统分页（`n-pagination` 页码组件）
- API 不返回 `total` → 无限滚动（`error` 字段表示结束）
- crawler 的 page 参数从 1 开始，内部兼容转换

## 样式原则
- **禁止写新样式之前不查全局已有的** — 先 grep `app.css` 和现有组件
  - `.jmz-panel` / `.jmz-panel--pad`：面板容器
  - `.jmz-chip` / `.jmz-chip--click`：标签/点击式标签
  - `.jmz-author-link`：作者链接
  - `.jmz-pager-footer` / `.jmz-pager-pagination` / `.jmz-pager-info`：分页底部
- **弹窗 = 页面组件套 Dialog** — MetaPage 同时用于 page 和 dialog 模式，不写两套
- **页面组件名统一用 Page 结尾**，路由 `/kebab-case`，nav 用对应图标
