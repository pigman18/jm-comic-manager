# JM漫画管理器

> 一个面向 JM 漫画的 **本地搜索与下载辅助应用**  
> ⚠️ **第三方个人项目，仅供学习与研究使用，严禁用于任何商业或非法用途**

---

## 🤔 为什么做这个应用

在长期使用网页版的过程中，总会遇到一些琐碎但影响体验的问题：

- 下载任务偶尔失败，便被遗忘在角落，时间一长也就不再想起去下载它；
- 下载到本地的漫画只剩下编号，难以检索以及还原作品名称；
- 想要根据特定 tag 筛选符合口味的漫画时，本地资源又几乎无从下手。

为了缓解这些碎片化痛点，这个应用将**下载、本地搜索与灵活配置**整合进来，让漫画检索与管理更加直观、高效。

---

## ✨ 功能简介

- ⬇️ **便捷下载**：支持章节级批量下载，大幅降低中断风险，并支持断点下载
- 📚 **本地搜索**：一键检索本地漫画资源，快速定位已下载作品
- 🧩 **灵活使用**：存在 core.js、core.exe、desktop.exe 三种不同形态，使用者可以根据不同需求使用成品
